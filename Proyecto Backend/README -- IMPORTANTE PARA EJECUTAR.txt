Pasos para ejecutar la aplicación Móvil

1. Descargar en BackEnd (proyecto_back_end_appmovil-master), descomprimir y ejecutar con spring boot
   el puerto de escucha es 8030

2. Descargar el FrontEnd (ReservasCDUAppMovil) e ingresar al archivo Constants que se encuentra ubicado en la carpeta java/co.edu.unicauca.lottieapp/utils/Constants y en la variable URL colocar la ip del computador en el que se va a ejecutar el proyecto, seguido del puerto del backend


3. Ejecutar el FrontEnd con Android Studio, y el jdk minimo es 7.0
